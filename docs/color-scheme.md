# Line Creek FSC Color Scheme

## Purple Theme (Figure Skating)

### Primary Colors
```css
--primary: #5B2C91        /* Deep purple - main brand color */
--primary-dark: #412069   /* Darker purple for contrast */
--primary-light: #7B4DB8  /* Lighter purple for accents */
```

### Secondary/Accent Colors
```css
--secondary: #E0BBE4      /* Light lavender - soft accent */
--accent: #D946EF         /* Bright purple/magenta - CTAs and highlights */
--ice-blue: #B8E6F0       /* Ice blue - complementary color */
```

### Neutral Colors
```css
--gray-900: #1a1a1a       /* Dark text */
--gray-800: #2d2d2d       /* Headers */
--gray-600: #6b7280       /* Body text */
--gray-300: #d1d5db       /* Borders */
--gray-100: #f3f4f6       /* Light backgrounds */
--white: #ffffff          /* White */
```

### Usage Guidelines
- **Primary Purple**: Navigation, headers, footers, major UI elements
- **Accent Purple**: Call-to-action buttons, links, highlights
- **Ice Blue**: Complementary accents, badges, secondary elements
- **Light Lavender**: Background tints, hover states, subtle accents
- **Neutrals**: Text, borders, backgrounds

## Comparison to Original Site
- Original: Purple tones with white backgrounds
- Modernized: Richer purple palette with better contrast and ice-themed accents
